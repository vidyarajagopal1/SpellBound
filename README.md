# SpellBound

A personal reading memory system built with Node.js, Express, and SQLite.

## Setup

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open `http://localhost:3000` in your browser.

## Features

- Manage books with status, category, and reflections.
- Capture highlights with reflections and impact levels.
- Browse books and highlights with filtering.

## Troubleshooting

- Ensure Node.js is installed.
- If port 3000 is in use, change it in `server.js`.
- Database file `spellbound.db` is created automatically.
